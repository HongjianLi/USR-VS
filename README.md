USR
===

This webserver is based on [istar] and implements the ultrafast shape recognition algorithms.


Architecture
------------

![istar architecture](https://github.com/HongjianLi/istar/raw/master/public/architecture.png)


Components
----------

### Web client

* [jQuery]
* [Twitter Bootstrap]
* [three.js]
* [jquery-dateFormat]

### Web server

* [node.js]
* [mongodb]
* [express]

### Database

* [MongoDB]

### Daemon

* [MongoDB C++ Driver]


Supported browsers
------------------

* Mozilla Firefox 41
