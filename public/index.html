USR-VS 2.0

large-scale ligand-based virtual screening powered by ultrafast shape recognition techniques

If the input file contains more than one molecule,
only the first molecule will be used.

Alternative molecular similarity methods usually lead to the discovery of different active molecules for the same target. USR has discovered previously unknown bioactive molecules for a range of targets. USRCAT has obtained substantially better retrospective performance.

The USR or USRCAT similarity between the query molecule and each of the 1B 3D conformers of 262M molecules will be calculated.
The user will be redirected to the result page with a unique URL.

Screening libraries

  • 269,411,840 molecules were collected from ZINC15.
  • 1,077,647,360 low-energy conformers were generated from these molecules using the ETKDGv2 algorithm implemented in RDKit 2019.03.2.
